Equation result as a float point type T.

Solves non-linear equation using Newton method. This function needs two functions, the function to be solved fun and its derivate dfun

Note
The function f(x) must be continues and differentiable.
Warning
Throws NonCoverge exception when the root is not found.
See Also
NewtonSolver
https://en.wikipedia.org/wiki/Newton%27s_method

Example:

// Solve f(x) = x^2 - 25.0 , df(x) = 2x around x0 = 10.0
auto fun = [](double x){ return x * x - 25.0 };
auto dfun = [](double x){ return 2 * x; }
double root = GenericNewtonsolver(fun, dfun, 10.0, 0.001);
std::cout << "Root = " << root << std::endl;
