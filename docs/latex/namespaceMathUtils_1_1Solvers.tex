Solves non-\/linear equation using Newton method. This function needs two functions, the function to be solved {\ttfamily fun} and its derivate {\ttfamily dfun} 

\begin{DoxyNote}{Note}
The function f(x) must be continues and differentiable. 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
Throws Non\-Coverge exception when the root is not found.
\end{DoxyWarning}

\begin{DoxyCode}
\textcolor{comment}{// Solve f(x) = x^2 - 25.0 , df(x) = 2x around x0 = 10.0}
\textcolor{keyword}{auto} fun = [](\textcolor{keywordtype}{double} x)\{ \textcolor{keywordflow}{return} x * x -  25.0 \};
\textcolor{keyword}{auto} dfun = [](\textcolor{keywordtype}{double} x)\{ \textcolor{keywordflow}{return} 2 * x; \}
 
\textcolor{keywordtype}{double} root = GenericNewtonsolver(fun, dfun, 10.0, 0.001);
std::cout << \textcolor{stringliteral}{"Root = "} << root << std::endl;
